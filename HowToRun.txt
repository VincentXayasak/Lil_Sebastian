--- GitHub Pages (Actions)
1) Repo Settings → Pages → set Source to GitHub Actions (deploy uses .github/workflows/deploy-pages.yml).
2) Settings → Secrets and variables → Actions → Repository secrets: SUPABASE_URL, SUPABASE_ANON_KEY (same as website/config.js locally or Supabase project settings API). Optional: PODCASTS_BUCKET, UPLOADS_BUCKET (default podcasts / uploads).
3) Push to main or Actions → Deploy website to GitHub Pages → Run workflow. After first deploy, note the Pages URL for Supabase Auth → URL configuration.
